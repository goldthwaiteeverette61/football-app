# ScoreRED 业务逻辑文档

## 🤖 AI挑选2串1方案

### 核心算法
1. **数据收集**
   - 历史比赛数据（3年+）
   - 球队技术统计
   - 球员状态数据
   - 实时赔率变化
   - 天气条件
   - 伤病报告

2. **AI预测模型**
   - 多层神经网络
   - LSTM时序分析
   - CNN图像识别
   - Attention权重分配

3. **2串1组合优化**
   - 筛选高置信度比赛（>75%）
   - 计算组合期望收益
   - 风险评估和排序
   - 返回前5个最优组合

### 预测准确率
- **单场预测**: 78-85%
- **2串1组合**: 65-75%
- **AI置信度**: 实时评估


#### 智能倍投策略
1. **动态倍投系数**
   - 基础系数: 1.5
   - 根据AI置信度调整
   - 连败次数影响系数
   - 最大系数限制: 3.0

2. **风险控制**
   - Kelly准则计算最优投注比例
   - 单次最大投注不超过剩余资金20%
   - 总风险控制在资金10%以内

3. **连黑理赔机制**
   - 连续失败8次自动触发
   - 理赔率: 80%
   - 保护用户资金安全
